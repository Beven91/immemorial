## immemorial

[![NPM version][npm-image]][npm-url]


### 简介篇

一个基于es7 `decorator`的代码通用生成工具。

###### 支持以下特性

- 可视化代码生成 （动态模板，在运行的基础上，进行代码生成）
- 通过`decorator`来控制生成数据以及生成行为
- 目前支持`React`模板生成 （后续会进行其他模式支持)

### 安装篇

    npm install immemorial
